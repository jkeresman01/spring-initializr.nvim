# Security Policy

## Supported Versions

At the current stage of the project, **only major releases** will receive security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.  
Instead, report it directly and privately to:  **josipkeresman@gmail.com**

## Disclosure Policy

- Vulnerabilities will be reviewed and fixed on a best-effort basis.  
- Fixes will be released in a patched version of the **latest major release** only.  
- Once a fix is released, details may be disclosed publicly.

*Thank you for helping make this project safer for everyone.*
